### Apollo Server Boilerplate

NodeJS apollo server boilerplate, the initial setup for a server:

#### Setup
Pull the repository and run
```
npm install
```

#### Run Server Command
```
npm start
```

#### Access GraphQL Playground at:
```
http://localhost:4000/graphql
```

#### Features:
- User Signup
- User Login
- User Roles
- JWT Token / Authorization
- Localization
